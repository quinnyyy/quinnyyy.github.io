Prim's Algorithm for Random Maze generation

  • Start with grid full of walls and a starting point (bottom middle)
  • Maintain a list of walls to look at. Start by adding the walls surrounding the starting point
  • Choose a random wall in the wall list. If the wall does not connect passages make it a passage. Add the surrounding walls to the wall list
  • Remove the chosen wall from the wall list
  • Repeat the following two steps until no walls remain in the wall list
  • Add an exit from a random point from the top row

Depth First Search

  • Choose a source (bottom middle)
  • Create a stack and add the source to the stack
  • While the stack is not empty pop the top off the stack and if it is not visited mark it as visited and add its neighbors to the stack.

Breadth First Search

  • Choose a source (bottom middle)
  • Create a queue and add the source to the stack
  • While the queue is not empty dequeue the first element of the queue and if it is not visited mark it as visited and add its neighbors to the queue.

A* Pathfinding Algorithm

  • Choose a source (bottom middle)
  • Create a collection of nodes under consideration beginning with the neighbors of the source
